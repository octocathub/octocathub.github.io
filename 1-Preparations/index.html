# 准备工作

TIP

任何语言的学习，提高，乃至成为“绝顶”高手，都是通过一行、一行又一行的代码敲击而成，都是通过不断地错误、不断地修改、不断地思考而完成的。只有这样，才能达到程序猿的最高状态：“静若瘫痪，动若癫痫”。

# 原材料🔨

在你阅读本文档之前，首先准备好一颗大脑，拥有耐心。
文档会很长，很多。你应该有足够的耐心去阅读。当然阅读并不是目的。目的在于你阅读之后可以了解更多，学到更多。
我们抱着学习的心态去阅读，这样才能走的更远。学习是个十分枯燥的过程，但是你只要拥有了兴趣，那就不一样了，这件事就会变得很有乐趣。你乐在其中，自然会学到更多。因为这不是逼迫你做的。
本文档涉及的所有内容，全部是我会的，或者是我正在学习中的。本文档会跟随着我学习的步伐一致处于更新状态。文档大部分都是以笔记的形式记录的。所以有时候看上去会很没有条里。但是只要记录了，只要学会了就好。

# 工具💻

一台Windows主机，一台Linux主机。并且全部配备有git。 在此声明一下，很多人不屑于使用Windows，甚至说Windows系统垃圾，不稳定，bug多。其实原因依然在于你不会用。你自认为图形化界面就是傻瓜式的，其实并不是那样。图形化界面的软件的设计的复杂程度是命令行界面的几百倍。(也许更多)。 每个操作系统都有其存在的价值，也从来不存在完美的操作系统。 所以，在此，我仍然提倡你使用Windows系统。并辅助以Linux系统进行开发。
